学习一波markdown语法

​ 刚建了属于自己的博客，心情还是很激动。既然有了博客，当然就要开始写了。写博客现在都是使用Markdown语法，大大加快效率，以前也知道一些Markdown,不过用的很少，很多都忘了，这次开始系统学习一下Markdown语法，写这个博客用来记录与监督自己的学习过程吧。

1 标题语法

​ Markdown支持两种标题的语法，Setext和atx形式。不过常用的还是Atx形式，在首行插入1个到6个#，对应标题标题1到6阶。就是#就代表<h1></h1>，2个就代表<h2></h2>，以此类推。

4 链接

Markdown支援两种形式的链接写法：行内和参考2种形式，都是使用角括号（[ ])来把文字转出链接。

行内形式是直接在后面用括号直接接上链接

This is a [blog](https://yyssll.top)

输出HTML为：

<p>this is an <a href="https://yyssll.top">blog</a></p>

你也可以选择加上title属性：

This is a [blog](https://yyssll.top/"with a Title").

5 图片

插入图片的写法与插入链接的写法非常类似，只需要在引用链接的格式下的前面加上一个！

![Alt text](图片地址)

​ 方括号里是图片的Alt内容，是图片未加载出来对图片的说明，可以不写。这种引入图片的缺点就是不能定义图片宽高，如果你需要的话，可以使用<img/>标签。

<img src = '地址' width ="100" height = "100" alt = "example"
