面试题html/css

发表于 2018-07-30 | 分类于 css/html | 热度: °C

部分html、css面试题（未待完续）

1.游览器页面由哪三层构成，分别是什么，作用是什么？

我的回答：由html,css,js构成，html作用是构建网站的骨架，css是给网站添加样式，js是给网站添加行为，与用户实现交互。

答案：结构层，表示层，行为层，分别是html，css，js，作用：html实现页面结构，css完成页面样式和风格，js实现一些功能和业务。

2.h5的优点与缺点？

我的回答：h5提供了很多功能强大的api，和新特性，使一些功能实现起来更加容易，减少了代码量，以及提供了很多新的标签，可以使html书写更加语义化，但是它做大的缺点就是对游览器门槛较高，很多低版本游览器不兼容。

答案：优点：1.标准统一，本身就是w3c提出来的 2.及时更新 3.提供很多新标签，可以使开发人员书写更有语义化的标签

4.很好的替代了Flash，Silverlight 5.被广泛用于移动应用程序和游戏 6.涉及到网站的抓取和索引的时候，更有利于seo 7.提供了更多的媒体元素（video，audio）6.多设备跨平台.

缺点：1.兼容问题，ie9以前全军覆没 2. 性能低下，某些平台上的引擎性能问题，导致html5性能低下。

  1. 完善性，许多特性各游览器支持程度不一样 4. 安全问题 web storage、web socket这些功能很容易被黑客利用，来盗取用户的信息和资料。
3.DocType的作用？严格模式与混杂模式如何区分？他们有何意义？

答案：1.DocType的位于文档的最前面，标签之前，是告知游览器的解析器，用什么文档类型规范来解析这个文档。

2.严格模式的排版和js运作模式是以该游览器支持的最高标准运行。

3混杂模式，页面以宽松的向后兼容的方式显示，模拟老式游览器的行为以防站定无法工作

4.DocType不存在，或者格式不正确，将导致文档以混杂模式呈现。

4.谈谈你对HTMl5的认识？（是什么，为什么）

答案：1.HTML5是html，css，js的一套技术组合，它增强了游览器的原生功能，减少了web引用对插件的依赖，让用户体验更好，让开发者更加方便。

5.cookie、localStorage、sessionStorage的区别？

答案：三者都是在游览器端存储数据，但是cookie可以将数据随请求发送到客户端，cookie存储量有限，每个域名只能存储大约4kb，所用域名存储量也有限制，大概4k左右。

另外2个功能大致一样，唯一的区别就是sessionStorage是临时存储，窗口一旦关闭就会被销毁。存储大小没有限制，localStorage是永久存储，单个域名存储量较大，存储5M左右的数据。总体数量没有限制。
