flex

发表于 2018-07-30 | 分类于 css | 热度: °C

1.设置flex布局后，子元素的float、clear、vertical-align属性将失效。

2.设置flex布局的元素称为容器，子元素称为项目。容器默认存在2根轴，水平方向默认为主轴（main-axis），垂直方向默认为交叉轴（cross-axis)。主轴方向可以有容器上的flex-direction属性改变。

3.容器上可以设置6个属性：flex-direction、flex-wrap、flex-flow、justify-content、align-items、align-content。

4.flex-direction属性是设置主轴的方向，即项目的排列方向，可设置的值有4个，row，row-reverse、column、column-reverse

  • row 默认值，主轴为水平方向，起点为左端
  • row-reverse ，主轴为水平方向，起点为右端
  • column，主轴为垂直方向，起点在顶端
  • column-reverse，主轴为垂直方向，起点在顶部

5.flex-wrap决定项目是否换行，可取3个值wrap（换行），no-wrap（默认，不换行），wrap-reverse（第一行在下方）

6.flex-flow是flex-direction和flex-wrap的缩写，默认值为row nowrap

7.项目上有6个属性

  • order 定义了项目的排列顺序，数值越小，排列越靠前
  • flex-grow 定义了项目的放大比例，默认值为0，即即使有剩余空间也不会放大，
  • 如果所有项目的flex-grow属性都为1，则他们将等分所有剩余空间（如果有的话），如果一个项目的flex-grow为2，其余的为1，那么前者占据的剩余空间将是后者的2倍
  • flex-shrink定义了项目的缩小比例，默认值为1，即空间不足都将等比例缩小，如果一个项目的flex-shrink为0，其他项目为1，空间不足吗，该项目不缩小，负值对该属性无效

8.flex属性

flex是flex-grow、flex-shrink、flex-basic的缩写形式。如果以上3个属性同时去取默认值，那么flex值为0 1 auto，同理，如下是等同的。

当flex取值为none是，则计算为flex:0 0 auto

当flex取值为auto是，则计算为flex: 1 1 auto`

当flex取值为一个非负数时，flex:1 1 0%,如最常见的flex:1

当flex取值为一个长度或者百分比时,则视为flex-basic的值，其余2个值为1

当flex取值为2个非负数字是，视为flex-grow,flex-shrink的值，flex-basic的值为0%

当 flex 取值为一个非负数字和一个长度或百分比，则分别视为 flex-grow 和 flex-basis 的值，flex-shrink 取 1

主轴上父容器的总尺寸为600px，子元素的总基准值为0%+auto+200px

故剩余的空间为600px-300px=300px,再根据各个子元素的放大比例来分配剩余空间，grow和为2+2+1，每一份为60px，

item1的flex-grow值为2，所以最终宽度为120，item2宽度为100px+60px*2为220px，item3为200px+60px为260px
